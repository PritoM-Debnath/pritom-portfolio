Research Experience

  1. Human Activity Recognition in Healthcare Systems

    Supervisor: DR. MD. KABIRUZZAMAN   Role: Researcher   Duration: June 2022 - Nov 2023

    Description: Conducted research on human activity recognition within nurse care systems, focusing on time-series classification of care records data. The study addressed critical challenges such as class imbalance, user variability, and signal noise in real-world healthcare environments. Supervised learning models—including Decision Trees, Random Forests, and SVMs—were evaluated across 28 activity classes derived from 38,000+ annotated care-record samples collected over four months.

    Achievements:

    • • Published as “Time Series Analysis of Care Records Data for Nurse Activity Recognition in the Wild” (CRC Press, Taylor & Francis).
    • • Demonstrated that Random Forests provided the most robust cross-user performance, while Decision Trees achieved the highest accuracy (~66%) under controlled distributions.
    • • Highlighted the impact of data imbalance and annotation noise on classifier stability, contributing to methodological improvements in healthcare activity recognition.

    Skills: Data preprocessing, feature extraction, time-series analysis, machine learning model development (Decision Trees, Random Forests, SVM), and empirical evaluation.

  2. Self-Adaptive User Interfaces for Elderly Users

    Supervisor: K. M. IMTIAZ-UD-DIN   Role: Undergraduate Researcher (AIUB)   Duration: Jan 2023 - July 2024

    Description: Investigated dynamic UI adaptation strategies for elderly users using Bayesian probabilistic modeling. Designed and implemented a self-adaptive interface capable of real-time adjustments of text and icon sizes based on user interaction features such as error count, zoom frequency, and gaze direction. The model operated in a two-phase cycle—baseline collection (“cold start”) and adaptive response (“adaptation state”) to continuously refine user accessibility.

    Achievements:

    • • Developed a pluggable Bayesian model that enhanced usability and accessibility for older adults with visual impairments.
    • • Demonstrated adaptive scaling behavior, with a 25% increase in element size triggered after repeated user interactions.
    • • Published in Proceedings of the 3rd International Conference on Computing Advancements (ICCA `24).

    Skills: Bayesian modeling, human-computer interaction (HCI), user experience design, adaptive interface systems, and accessibility research.
